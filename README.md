
# SQLiteUsers (PDDM)

Ejercicio: Login + UserData + Manage Users + Backup/Restore con SQLite.

Funcionalidades
- Login contra SQLite (tabla Usuarios)
- Pantalla User Data
- Gestión de usuarios (alta/baja)
- Crear backups de la BD en /Android/data/<paquete>/files/backups
- Restaurar la BD desde un backup seleccionado

Requisitos
- Android Studio
- SDK actual

Ejecución
Abrir el proyecto en Android Studio y ejecutar en emulador/dispositivo.
